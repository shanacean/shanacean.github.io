CSS学习总结

Cascading Style Sheets(层叠样式表)是用来控制网页在浏览器中的显示外观的声明式语言。浏览器会根据 CSS 的样式定义将其选定的元素显示为恰当的形式。一条 CSS 的样式定义包括属性和属性值，它们共同决定网页的外观。

一、CSS基础分析

1.添加css的三种方式

  • 外部样式表(在<head>标签中添加)
  • 内部样式表(在<head>中直接写入样式)
  • 内联样式表(在需要添加样式的标签上直接写入)

    实际中主要以外部样式表为主，低耦合并且在多处HTML文件中可用。

2.CSS语法

选择器 + 以 =={}== 包含的多个属性-值对，每一对以 ==;== 结尾。

3.三种主要选择器

  • 标签选择器

  • id选择器

  • 类选择器

4.盒子模型

盒子组成：

  • Content box: 显示内容
  • Padding box: 内边距
  • Border box: 边框
  • Margin box: 外边距

一个元素占据的宽度为lmargin + lborder + lpadding + content + rpadding + rborder + rmargin
高度同理
height、width只是设置内容的高度、宽度

5.定位、溢出、浮动、不透明度

  • position
    • static： 默认html布局位置
    • relative： 相对于默认的位置偏移
    • fixed：使元素固定不动
    • absolute：使元素相对于非static的父元素偏移
  • overflow
    • visible
    • hidden
    • scroll
    • auto
  • float
    • left
    • right
    • inline-start
    • inline-end
  • opacity

    在[0-1.0]之间

二、发掘

1.border-radius 圆角

  • 四个值: 左上角，右上角，右下角，左下角。
  • 三个值: 左上，右上左下，右下
  • 两个值: 左上右上，左下右下
  • 一个值：四边同

2.transition 过渡

类似于动画效果，可指定要添加效果的元素和过渡的持续时间
