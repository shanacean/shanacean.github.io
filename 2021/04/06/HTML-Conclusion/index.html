HTML学习总结

一、主体框架分析

HTML(Hyper Text Markup Language) 超文本标记语言，主要用于网页内容的显示，配合css, javascript食用。
主体框架如下:

  • <head>
    主要存放一些元数据，以及 title、link、script 等信息…

    • <title>
        页面标题，显示在窗口选项卡中。
      • <meta> 具有以下属性：
        • name 提供文档级别元数据，应用整个页面
        • http-euqiv 编译指令
        • charset 字符编码设置
        • itemprop 用户自定义元数据

    每个 <head> 只能包含一个title元素

  • <body>
    存放网页内容信息

二、文字基础

<p> : paragraph

段落标签，表示文本中的一个段落

<h1> : head

标题标签, 级别为1-6

<ul> : unordered list

无序列表

<ol> : ordered list

有序列表

列表可嵌套使用

<em> : emphasis

强调，斜体字符

<strong> : strong importance

强调，粗体字

<dl>  : description list

描述列表

  • <dt>  description terms

    描述术语

  • <dd>  description description

    描述部分

<blockquote>

块引用，通常会进行缩进，可以将引用处URL地址设置在 ==city==上，也可以用 <city>元素以文本形式写出出处。

  • <q> : quote

    行内引用

三、超链接

<a> : anchor

锚点，可以创建通向其他网页、文件、同一页面内的位置、电子邮件地址或任何其他 URL 的超链接。

注意绝对路径，相对路径的问题。

四、图片、视频

  • 添加图片
  • 图片搭配文字

五、表格

<table>

  • <tr> table row
  • <th> table head
  • <td> table data
  • <col> column
  • <clogroup> column group
  • <caption>

colgroup 给表格中的一组colunm设置格式

col为span选中的列设置格式

caption为表格添加标题
